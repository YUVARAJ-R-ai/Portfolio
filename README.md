# The Grand Line Portfolio

This is a 3D interactive portfolio website inspired by the world of One Piece. It uses Three.js to create a globe that users can interact with.

## Technologies Used

*   HTML
*   CSS
*   JavaScript
*   Three.js
*   Tailwind CSS

## How to Use

1.  Clone the repository.
2.  Open the `index.html` file in your browser.

## Features

*   Interactive 3D globe
*   Hover over the islands to see project details
*   Click the "Start Journey" button to take a tour of the projects
